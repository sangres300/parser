# Парсер оценок пользователей Киноафиша

## 1. Постановка задачи
Проект предназначен для сбора данных об оценках фильмов конкретным пользователем сервиса kinoafisha.info. 
Это необходимо для последующего построения рекомендательных систем на основе предпочтений пользователей.

## 2. Инструкция по сборке и запуску
1. Установите Python версии 3.8 или выше.
2. Установите библиотеку BeautifulSoup: `pip install beautifulsoup4 requests`.
3. Запустите скрипт: `python kino_parser.py`.

## 3. Результаты
Программа сохраняет данные в файлы `results_ID.json` и `results_ID.csv` в корневой папке проекта.
